Инстанс axios - это экземпляр объекта, созданного из библиотеки Axios, 
которая предоставляет удобные методы для отправки HTTP-запросов из JavaScript.

Создание инстанса axios позволяет настроить параметры запросов, такие как базовый URL, 
заголовки и обработчики ошибок. Это позволяет упростить отправку запросов в различных 
частях приложения, таких как компоненты React или модули Node.js.
